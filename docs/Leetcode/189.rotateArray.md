```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        l = len(nums)
        k = k % l
        res = nums[l-k:] + nums[:l-k + 1]
        for i in range(len(nums)):
            nums[i] = res[i]
```
